# Medical Side Effect Tracker

A Python console application that allows users to log medication side effects, record severity levels, save entries to a CSV file, and generate simple summary reports for tracking trends over time.

## Features
- Add medication side effect entries
- Record severity levels from 1 to 5
- Save entries to a CSV file
- View all logged entries
- Generate a summary of common side effects and average severity by medication

## Technologies Used
- Python
- CSV file handling
- Basic data validation
- Functions and loops

## How to Run
1. Download or clone the repository
2. Open the project in PyCharm or another Python IDE
3. Run `main.py`
4. Use the menu options to add entries, view entries, or show a summary

## Example Use Case
This project can be used as a simple symptom and side effect logging tool for personal tracking and basic trend analysis.

## Project Purpose
This project was created to demonstrate Python programming concepts including file handling, user input validation, looping, functions, and basic data analysis in a healthcare-related context.
